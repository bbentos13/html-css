Inspecionando Sites

Vamos testar seus conhecimentos com algumas perguntas:

1) Em CSS, existem vários símbolos que representam os tipos de seletores que temos. O símbolo # especifica qual seletor ?

    a) Class
    b) Tag
    c) ID
    d) First
    e) Body

R: ID

2) O que não é um elemento HTML5?

    a) Section
    b) Header
    c) Blink
    c) Body
    d) Main

R: MAIN

3) O que é Front-End?

    a) É uma profissão
    b) É a ponte entre o banco de dado e interface aplicando as devidas regras de negócio
    c) É o desenvolvimento da interface gráfica do usuário
    d) É um framework
    e) É a tecnologia utilizada para criar um banco de dados

R: É O DESENVOLVIMENTO DA INTERFACE GRÁFICA DO USUÁRIO

4) Qual alternativa abaixo NÃO é uma tecnologia utilizada no Front-End ?

    a) ReactJS
    b) VueJS
    c) SQL
    d) AngularJS
    e) Bootstrap

R: SQL

5) Temos a seguinte declaração de variável "var a = [2,3,4]; ", qual o tipo da variável "a" ?

    a) String
    b) Object
    c) Array
    d) Hashmap
    e) Int

R: ARRAY

6) Uma página web só pode ter um único elemento com ID com o mesmo nome

    a) Verdadeiro
    b) Falso

R: VERDADEIRO

7) Qual propriedade CSS é usada para mudar a cor de um elemento de texto ?

    a) Fontcolor
    b) Color
    c) Textcolor
    d) Color-text
    e) Text: #ccc

R: COLOR

8) Qual elemento HTML é utilizado para colocar código Javascript ?

    a)Script
    b)Js
    c)JAVASCRIPT
    d)Link
    e)Java

R: SCRIPT

9) Definindo a margem dessa forma "margin: 5px 10px 3px 8px;" qual o valor dado para a margem esquerda ?

    a) 5px
    b) 10px
    c) 3px
    d) 15px
    e) 8px

R: 8PX

10) O que é o REACT ?

    a) Um framework
    b) Uma biblioteca
    c) Um livro
    d) Uma liguagem
    e) Nenhuma das alternativa acima

R: UMA BIBLIOTECA
